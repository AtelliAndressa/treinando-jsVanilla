Questões e possibilidades

Responda quais são as diferenças entre os elementos 'div’ e ‘span’ na construção de páginas HTML.
O div é usado para agrupar elementos para fim de estilo baseado em blocos. O span é usado para agrupar elementos para fim de estilo baseado em linhas. Só deve ser utilizado quando nenhum outro elemento semantico for possível.

Por que não se aconselha o uso indiscriminado de elementos ‘div’?
O "div" não dá nenhuma informação útil sobre a estrutura de um documento, e os robos não conseguem ainda fazer a distinção em sua leitura. Para ler o código, você precisa verificar cuidadosamente os nomes das classes, torna-se complicado controlar quais tags de fechamento acompanham as tags de abertura. Uma maneira padronizada de marcar estruturas comuns em documentos da web deixaria bem mais fácil o trabalho de ler o código.

Explique quais seriam as alternativas para reduzir o uso destes elementos em div:
- O section agrupa conteudo por tema e representa uma seção. Podem ter vários por páginas;
- O main especifica a seção principal do conteúdo. Deve ser usado uma única vez por página;
- O aside representa parte de uma pagina que contem conteúdo relacionado a uma determinada seção;
- O article separa seções de conteudo que podem ser independentes, como postagens em blogs, jornais etc;
- O blockquote representa um conteudo que está sendo citado de uma fonte externa, frequentemente vem acompanhado do cite;
- O nav representa a seção de navegação de um documento;
- O footer representa o rodapé de uma seção.

Qual a finalidade do elemento de entrada do tipo ‘hidden’? Explique.
A finalidade é ocultar os dados adquiridos no input, isto fica oculto ao usuário e visível apenas ao desenvolvedor.
